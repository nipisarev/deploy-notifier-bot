# Deploy-Notifier-Bot
The tool for notification you deployment processes to slack, hipchat and etc.

## Get Started
### Run
For simple run project and start listen on [localhost:8181](http://localhost:8181)
```bash
go run *.go
```
Or build 
```bash
go build
```
